Ejemplo de utilización de Power Bi Desktop para el análisis de un conjunto de datos del propio Power Bi. 

Se realizan diferentes gráficos, como el beneficio obtenido según el avance del tiempo, según el país y también las ventas según el segmento y el producto. 

Además permite la interactividad, ya que se permite seleccionar los años o los meses que pueden resultar de interés.
